# Store 画像素材チェック

Microsoft Store 申請前に、ここへ公開用スクリーンショットや追加画像を置いて確認します。
個人のワークスペース名、ユーザー名、パス、APIキー、未公開コード、社内情報が写らない画像だけを使ってください。

## 必須

- デスクトップ用スクリーンショットを最低1枚用意する。
- 推奨は4枚以上。
- PNG形式にする。
- 1366 x 768 px 以上にする。
- 1ファイル50 MB以下にする。

## 推奨ファイル名

- `desktop-01-main.png`: 4面表示の全体像
- `desktop-02-compact.png`: 縮小モード
- `desktop-03-status-frame.png`: AI状態フレーム
- `desktop-04-settings.png`: 設定・保存まわり

## 撮影前チェック

- サンプル用の空フォルダやダミーリポジトリを開く。
- VS Code のターミナル、エディタ、最近開いた項目に個人情報がないことを確認する。
- Store掲載文案と矛盾する機能を写さない。
- Microsoft、Visual Studio Code、GitHub、OpenAI、Anthropic の公式アプリであるように見える表現を避ける。
