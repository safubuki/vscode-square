# サポート案内草案

Turtle AI Code Quartet Hub の公開前サポート文書です。
Microsoft Store 提出前に、正式なサポートURL、連絡先、対応範囲に置き換えてください。

## 対応範囲

- アプリの起動、設定、VS Code ウィンドウ配置に関する問題。
- スロット A-D の保存、復元、控え Quartet に関する問題。
- AI 状態フレームの表示や更新に関する問題。
- Microsoft Store 版のインストール、更新、アンインストールに関する問題。

## 対応外

- Visual Studio Code 本体の不具合。
- VS Code 拡張機能、AIサービス、Remote SSH、GitHub Copilot、Codex 側の障害。
- ユーザーのプロジェクト、リポジトリ、ソースコードの内容に関する調査。

## 問題報告に含める情報

- Windows のバージョン。
- Turtle AI Code Quartet Hub のバージョン。
- Visual Studio Code のバージョン。
- 使っている AI 拡張機能名。
- 問題が起きた操作手順。
- 個人情報や機密情報を隠したスクリーンショット。

## ローカルデータの削除

アプリの設定や実行時データを削除するには、次のフォルダを削除してください。

```text
%LOCALAPPDATA%\TurtleAIQuartetHub\
```

## 連絡先

正式公開前に、公開者のサポートURLまたは連絡先へ置き換えてください。
