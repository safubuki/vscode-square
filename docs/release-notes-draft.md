# リリースノート草案

Microsoft Store 提出時の「このバージョンの新機能」や、GitHub Releases 用の下書きです。

## 1.0.0

初回公開候補。

- 4つの Visual Studio Code ウィンドウを A-D スロットとして起動・配置。
- 2x2 の4面表示と、常時最前面の縮小モードに対応。
- VS Code 外周の状態フレームで AI 実行中、完了、確認待ち、待機中を表示。
- スロット名、ワークスペース、控え Quartet、レイアウト状態をローカル保存。
- スロット別 VS Code user-data-dir により、複数セッションを分けて管理。
- ローカルの VS Code UI と一部拡張ログから AI 状態を推定。
- アプリ独自のテレメトリ送信なし。

## 提出前に確認すること

- バージョン番号がパッケージと一致していること。
- Store掲載文案と機能説明が一致していること。
- スクリーンショットに個人情報や未公開コードが写っていないこと。
- プライバシーポリシーURLとサポートURLが有効であること。
